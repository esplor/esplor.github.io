iPXE

You can chainload your iPXE to this file for a Debian / Ubuntu network install.

  • iPXE undionly (PXE chainload)
  • iPXE iso file
  • iPXE efi 64 boot file

Remember:

You have to be in the src directory of iPXE.

cat <<EOF> esplor.github.io.boot.ipxe
#!ipxe

dhcp
chain http://esplor.github.io/boot.ipxe
EOF

make bin/undionly.kpxe EMBED=esplor.github.io.boot.ipxe

If you don't want too make yourself a pxe file, you may use my ISO file. Dowload, burn it or mount it. Iso file is tested in VirtualBox and VmWare.

Note:

Debian use http://httpredir.debian.org to redirect to your closest mirror.

Source code at https://github.com/rgeissert/http-redirector

Ubuntu use the official Norwegian mirror. (http://no.archive.ubuntu.com)

How to add Virtualbox to a Debian box..

Become root/superuser first: "sudo -i" or "su -" 
cat <<EOF> /etc/apt/sources.list.d/vbox.list
deb http://download.virtualbox.org/virtualbox/debian wheezy contrib
EOF
wget -q http://download.virtualbox.org/virtualbox/debian/oracle_vbox.asc -O- | apt-key add -
apt-get update
apt-get install virtualbox-4.3
